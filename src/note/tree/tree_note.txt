307. Range Sum Query - Mutable  segment_tree
  Given nums = [1, 3, 5]

  sumRange(0, 2) -> 9
  update(1, 2)
  sumRange(0, 2) -> 8

  For each node at index i, the left child is at index 2*i+1, right child at 2*i+2 and the parent is at (i-1)/2.

  the total number of nodes will be 2*n-1, n leaves and n-1 interval nodes.
  1 Find the sum of elements from index l to r where 0 <= l <= r <= n-1

  2 Change value of a specified element of the array to a new value x. We need to do arr[i] = x where 0 <= i <= n-1.

  both above operations in O(logn) time
